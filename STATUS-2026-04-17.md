## What Is This Project?

**The Pour List** — a crowd-sourced happy hour directory. Users scan happy hour menus with their phone camera, AI extracts the text, and the parsed menu is stored permanently (photo discarded). No accounts. Anonymous device hash for spam prevention. Designed to scale nationally.

## Moderation Rules (FINALIZED + BUILT)

| Rule | Value |
|------|-------|
| Flag GPS required | YES — 10m Haversine |
| Confirm GPS required | YES — 10m Haversine |
| Flagger can confirm | NO |
| Flagger can reopen own close | NO |
| Re-flag after decay | BLOCKED — once flagged, never again |
| Flags per device per venue | 1 per day (enforced in DB) |
| Flags before submission | Requires 1 successful submission first |
| Flag decay | 1 flag removed per month per venue |
| Menu commit | Clears all flags on venue |
| Trusted threshold | 10 submissions → 'trusted' |
| Trusted flag weight | 2x toward N threshold |

### Threshold
- N=2 distinct devices → status: `'stale'` (hidden from main map)
- N=4 distinct devices → status: `'closed'` (fully hidden)
