# The Pour List

**Find and share happy hour menus in Portland, Oregon.**

The Pour List is a community-driven guide to the best drink deals in the city. Snap a photo of any happy hour menu, and it gets added to the map — permanently. No account needed.

---

## Quick Start

### Find Happy Hours

1. Open the app — it loads the Pearl District by default
2. Switch between **Map** and **List** view using the tabs
3. **Map pins** are color-coded:
   - 🟡 Amber = has a menu on file
   - 🟣 Purple = **happy hour active right now**
4. Tap any pin to see the venue's menu, hours, and directions

**Tip:** Use the radius slider on the right side of the map to expand or shrink the search area.

### Scan a Menu (Add or Update)

Tap the big orange button at the bottom of the screen:

> 📷 Scan Happy Hour Menu / Add Venue

From there:

1. **Take photos** — 1 to 4 photos of the menu (both sides of a two-page menu count as 2 photos)
2. **Confirm the venue** — if GPS is on, the app asks "Are you at [Venue Name]?" with a Yes/No. If there's no match, you can type the name and the app will check for duplicates nearby
3. **Review the text** — the app reads your photo(s) and shows the extracted menu text. You can edit it before saving
4. **Save** — the menu is permanently stored and visible to everyone

### Add a Venue (No Photo)

If a venue isn't on the map yet but you don't have a menu photo:

1. Tap the scan button
2. Tap "Take Photos / Choose from Gallery" but cancel before selecting files, then look for the "Add a Venue" option — or ask a developer to add this shortcut 😅

---

## How It Works

- **No accounts.** Your device generates a anonymous fingerprint hash (`device_xxxxx`) that's used for spam prevention. Nothing is linked to your identity.
- **GPS pinning.** If your photos have GPS data in them, the venue gets pinned automatically. No manual address entry required.
- **Community moderation.** Venues start as `unverified`. If enough people flag bad data, it gets reviewed.
- **Photo retention.** Each venue keeps the 3 most recent photos. Older ones are automatically deleted.

---

## Key Concepts

| Term | What it means |
|------|---------------|
| **Active HH** | A purple pin means the app thinks happy hour is running *right now* based on the stored menu text |
| **unverified** | New venue, added by a community member — hasn't been manually confirmed yet |
| **verified** | Confirmed to exist and have a HH program |
| **stale** | Had a menu before, but it hasn't been updated in a long time |
| **closed** | The venue is no longer operating |
