## DATA FRAMEWORK (for database schema)

Each venue record should include:
- `name` — venue name
- `address` — street address
- `city` — Portland
- `state` — OR
- `zip` — 97209
- `phone` — phone number
- `website` — URL
- `type` — bar, restaurant, cocktail lounge, etc.
- `happy_hour_start` — time (may differ by day)
- `happy_hour_end` — time
- `happy_hour_days` — array: ["mon","tue","wed","thu","fri","sat","sun"]
- `late_night_start` — second happy hour window if applicable
- `late_night_end` — second happy hour window end
- `deals` — JSON: { drinks: [], food: [], notes: "" }
- `photos` — array of user-submitted photo URLs
- `last_verified` — date
- `source` — where data originated
- `status` — "active" | "closed" | "unverified"
